CARGAR UN BLOB EN AZURE STORAGE
===============================

Una vez creado un contenedor, podemos cargar blobs. 

Utilizando AZURE PORTAL:

1. Azure Portal
2. Blob Service / Contenedor --> Seleccionamos nuestro contenedor

● Si creamos la cuenta de almacenamiento con soporte para nombres jerárquicos (para Data Lake Storage),
  la sección "Blob service" no aparece en el menu. En su lugar, seleccionamos "Contenedores" debajo de
  Data Lake Storage. Esto porque el Data Lake Storage es una "extensión de Azure Blob Storage", el
  servicio se construye sobre Azure Blob Storage.

En la página del Contenedor:

1. Cargar (Upload)
2. Cargar Blob -----------> UPLOAD BLOB 
                            - Seleccionar archivo
                            - Avanzadas 
                              - Tipo de blob
                              - Tamaño de bloque
                              - Nivel de acceso
                              - Cargar en carpeta
                              

El URI de un blob es similar a:

https://myaccount.blob.core.windows.net/mycontainer/myblob

o

https://myaccount.blob.core.windows.net/mycontainer/myvirtualdirectory/myblob
